# RabbitMq消息队列

## 1. 简介

RabbitMQ是一个实现了AMQP（Advanced Message Queuing Protocol）高级消息队列协议的消息队列服务，用Erlang语言。

[详细介绍](https://zhuanlan.zhihu.com/p/63700605)
